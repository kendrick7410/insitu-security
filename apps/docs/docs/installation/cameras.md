# Installation des caméras

Guide d'installation pour les caméras InSitu Security (intérieur et extérieur).

## Caméra Intérieure HD

### Contenu

- 1x Caméra intérieure 1080p
- 1x Adaptateur secteur USB
- 1x Support de fixation
- 2x Vis et chevilles

### Emplacement recommandé

- **Salon** : angle de pièce pour vue large
- **Entrée** : face à la porte d'entrée
- **Couloir** : zones de passage
- **Chambre enfant** : pour monitoring (optionnel)

:::tip Astuce
Utilisez la **visualisation AR** depuis notre site pour trouver l'emplacement parfait !
:::

### Installation

1. **Branchez la caméra** à une prise
2. **Ouvrez l'app** InSitu Security
3. **Appuyez sur "+"** puis "Ajouter une caméra"
4. **Scannez le QR code** au dos de la caméra
5. **Suivez le jumelage** (voyant clignote en bleu)
6. **Nommez la caméra** (ex: "Salon")

### Réglages recommandés

- ✅ Activer la vision nocturne automatique
- ✅ Détection de mouvement ON
- ✅ Zone de détection personnalisée (éviter fenêtres)
- ✅ Enregistrement sur événement
- ❌ Enregistrement continu (consomme beaucoup de stockage)

## Caméra Extérieure 4K

### Spécificités

- Étanche IP67
- Vision nocturne couleur
- Projecteur LED intégré
- Détection IA personne/véhicule

### Emplacement recommandé

- **Entrée principale** : face à la porte
- **Garage/Allée** : couvre accès véhicule
- **Jardin** : zones sensibles
- **Portail** : surveillance accès

:::warning Important
La caméra doit être sous un **avant-toit** ou protégée par le boîtier IP67 fourni.
:::

### Installation

1. **Fixation murale**
   - Marquez les 4 trous de fixation
   - Percez (mèche béton si nécessaire)
   - Vissez le support

2. **Câblage**
   - Option A : Prise extérieure étanche
   - Option B : Câble passant par un trou vers intérieur

3. **Jumelage**
   - Même procédure que caméra intérieure
   - Scannez le QR code au dos

4. **Orientation**
   - Angle vers le bas (15-30°)
   - Éviter le soleil direct dans l'objectif
   - Tester la vue dans l'app

### Réglages spécifiques extérieur

- ✅ Détection IA (personnes uniquement, ignore animaux)
- ✅ Projecteur automatique la nuit
- ✅ Réglage de sensibilité (éviter fausses alertes vent/pluie)
- ✅ Enregistrement cloud (recommandé)

## Configuration avancée

### Zones de détection

1. Dans l'app, allez sur la caméra
2. Paramètres > Détection de mouvement
3. Dessinez les zones à surveiller
4. Exclure : fenêtres, arbres, routes

### Enregistrement

**Options de stockage :**

- **Local (carte microSD)** : Gratuit, 7 jours
- **Cloud Basic** : 30 jours, 3 caméras - 4,99€/mois
- **Cloud Premium** : 60 jours, illimité - 9,99€/mois

### Notifications

- Snapshot inclus dans la notification
- Délai de notification : 2-5 secondes
- Mode silencieux (désactive la nuit)

## Qualité vidéo

| Résolution | Usage | Bande passante |
|-----------|-------|----------------|
| 4K | Extérieur, détails max | 8 Mbps |
| 1080p | Intérieur, standard | 2 Mbps |
| 720p | Économie données | 1 Mbps |

:::tip Conseil
Utilisez **1080p en continu** et **4K sur événement** pour équilibrer qualité/stockage.
:::

## Dépannage

### Image floue

- Nettoyez l'objectif
- Vérifiez la mise au point (auto normalement)
- Repositionnez pour éviter buée (extérieur)

### Décalage vidéo

- Vérifiez votre WiFi (signal > 70%)
- Rapprochez la caméra de la centrale
- Utilisez la 2.4GHz (meilleure portée)

### Vision nocturne ne fonctionne pas

- Vérifiez le mode automatique
- Nettoyez les LEDs infrarouges
- Réinitialisez la caméra (bouton Reset 10 sec)
