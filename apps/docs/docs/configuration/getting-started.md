# Premiers pas

Configuration initiale de votre système InSitu Security.

## Assistant de configuration

Lors de la première utilisation, l'app mobile vous guide à travers :

1. **Ajout de la centrale**
2. **Connexion WiFi**
3. **Création du compte**
4. **Ajout des appareils**
5. **Configuration de base**

## Paramètres essentiels

### Code PIN

- 4 à 8 chiffres
- Ne pas utiliser : 0000, 1234, votre année de naissance
- Changeable dans Réglages > Sécurité

### Délais d'armement

- **Sortie** : 30 secondes (temps pour quitter après activation)
- **Entrée** : 30 secondes (temps pour désactiver avant alarme)

Ajustez selon vos besoins dans Réglages > Alarme.

### Modes d'alarme

**Total**
- Tous les capteurs actifs
- Idéal quand vous partez

**Partiel**
- Périmètre uniquement (portes/fenêtres)
- Mouvements intérieurs ignorés
- Idéal quand vous êtes chez vous

**Nuit**
- Configuration personnalisée
- Ex: Rez-de-chaussée uniquement

## Ajout d'appareils

1. Ouvrez l'app
2. Appuyez sur **+** (en haut à droite)
3. Sélectionnez le type d'appareil
4. Suivez les instructions de jumelage
5. Nommez l'appareil
6. Assignez à une zone

## Test du système

Avant la première utilisation réelle :

1. **Testez chaque capteur**
   - Ouvrez/fermez portes et fenêtres
   - Marchez devant détecteurs mouvement
   - Vérifiez les notifications

2. **Testez la sirène**
   - Volume adapté ?
   - Durée correcte ?
   - ⚠️ Prévenez vos voisins !

3. **Testez l'armement**
   - Activez en mode Total
   - Déclenchez volontairement
   - Vérifiez délai d'entrée
   - Désactivez avec code PIN
